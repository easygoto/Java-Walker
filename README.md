# Java 学习之路

## 项目结构

```
<root>
 |-- config: 配置文件
 |-- images: 图片资源
 |-- src/main/java: 源码
     |-- core: 小工具
     |-- dp: 设计模式
     |-- gui: GUI应用
     |-- example: 实例
         |-- jdbc: 操作数据库
         |-- rabbit: 消息队列
         |-- security: 加密解密
         |-- springboot: 框架学习
         |-- thread: 多线程
 |-- src/test/java: 测试用例
```
